# 书潮 · 全栈书籍阅读平台

本项目已升级为全栈版本，包含前端阅读界面、后端 API、外部书源检索与书籍录入。

## 运行方式
```bash
cd d:\codex
npm install
npm run dev
```
浏览器访问：http://localhost:3000

## 功能概览
- 全站搜索 / 标签筛选 / 分区浏览
- 智能推荐与热度榜单
- 外部书源检索（公共 API 示例）
- 本地录入小说 / 名著 / 自有文本
- 阅读器视图与进度保存

## 项目结构
- `public/` 前端页面与样式
- `server.js` 后端服务与 API
- `data/library.json` 本地书库数据
- `data/sources.json` 外部书源配置

## 外部书源说明
已内置公共 API 示例：
- Open Library
- Project Gutenberg (Gutendex)

如需接入其他站点，请务必使用 **已授权** 的接口或数据源，避免抓取违反条款的数据。

## 录入书籍
前端“录入书籍”表单支持粘贴长文本，保存后即可在阅读器中阅读。

## 常见问题
- 若提示“API 未启动”，请确认 `npm run dev` 已启动后端。
- 若外部搜索失败，请检查网络或书源配置。
